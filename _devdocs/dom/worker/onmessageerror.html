Worker.onmessageerror

The onmessageerror event handler of the Worker interface is an EventListener, called whenever an MessageEvent of type messageerror is fired on the Worker instance â€” that is, when it receives a message that cannot be deserialized.

Note: This feature is available in Web Workers.

Worker.onmessageerror = function() { ... };
