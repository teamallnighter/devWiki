PaymentRequestEvent.topLevelOrigin

This is an experimental technology
Check the Browser compatibility table carefully before using this in production.

The topLevelOrigin read-only property of the PaymentRequestEvent interface returns the top-level origin where the PaymentRequest object was initialized.

Syntax

var aUsvString = paymentRequestEvent.topLevelOrigin

Value

A USVString
