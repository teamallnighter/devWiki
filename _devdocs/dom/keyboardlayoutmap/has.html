KeyboardLayoutMap.has

This is an experimental technology
Check the Browser compatibility table carefully before using this in production.

The has() method of the KeyboardLayoutMap interface returns a boolean indicating whether the object has an element with the specified key. A list of valid keys is found in the UI Events KeyboardEvent code Values spec.

Syntax

var aBoolean = KeyboardLayoutMap.has(key)

Parameters

key
The key of an element to search for in the map.

Return value

A Boolean indicating whether the specifed key was found.
