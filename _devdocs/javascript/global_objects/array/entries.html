array.entries

The entries() method returns a new Array Iterator object that contains the key/value pairs for each index in the array.

Syntax

array.entries()

Return value

A new Array iterator object.

Examples

Using a for…of loop

var a = ['a', 'b', 'c'];
var iterator = a.entries();

for (let e of iterator) {
  console.log(e);
}
// [0, 'a']
// [1, 'b']
// [2, 'c']
