padding-bottom

The padding-bottom CSS property sets the height of the padding area on the bottom of an element.

An element's padding area is the space between its content and its border.

Note: The padding property can be used to set paddings on all four sides of an element with a single declaration.

Syntax

/* <length> values */
padding-bottom: 0.5em;
padding-bottom: 0;
padding-bottom: 2cm;

/* <percentage> value */
padding-bottom: 10%;

/* Global values */
padding-bottom: inherit;
padding-bottom: initial;
padding-bottom: unset;

The padding-bottom property is specified as a single value chosen from the list below. Unlike margins, negative values are not allowed for padding.

Values

<length>
The size of the padding as a fixed value. Must be nonnegative.
<percentage>
The size of the padding as a percentage, relative to the width of the containing block. Must be nonnegative.

Formal syntax

<length> | <percentage>

Examples

.content { padding-bottom: 5%; }
.sidebox { padding-bottom: 10px; }
