@media.-moz-windows-default-theme

Non-standard
This feature is non-standard and is not on a standards track. Do not use it on production sites facing the Web: it will not work for every user. There may also be large incompatibilities between implementations and the behavior may change in the future.

Note: Since Firefox 58, this media feature is no longer available to web content — it is only available internally (e.g. to XUL / chrome code). See bug 1396066 for more details.

Syntax

<integer>
If the user is using one of the default Windows themes—Luna, Royale, Zune, or Aero (including Vista Basic, Vista Advanced, and Aero Glass)—this is 1. Otherwise it's 0.

Media: media/visual
Accepts min/max prefixes: no
