tab-size

This is an experimental technology
Check the Browser compatibility table carefully before using this in production.

The tab-size CSS property is used to customize the width of a tab (U+0009) character.

/* <integer> values */
tab-size: 4;
tab-size: 0;

/* <length> values */
tab-size: 10px;
tab-size: 2em;

/* Global values */
tab-size: inherit;
tab-size: initial;
tab-size: unset;
Initial value 8
Applies to block containers
Inherited yes
Media visual
Computed value the specified integer or an absolute length
Animation type a length
Canonical order the unique non-ambiguous order defined by the formal grammar

Syntax

Values

<integer>
The number of spaces in a tab. Must be nonnegative.
<length>
The width of a tab. Must be nonnegative.

Formal syntax

<integer> | <length>

Examples

pre {
  tab-size: 4; /* Set tab size to 4 spaces */
}

pre {
  tab-size: 0; /* Remove indentation */
}

pre {
  tab-size: 2; /* Set tab size to 2 spaces */
}
