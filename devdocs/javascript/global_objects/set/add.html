set.add

The add() method appends a new element with a specified value to the end of a Set object.

Syntax

mySet.add(value);

Parameters

value
Required. The value of the element to add to the Set object.

Return value

The Set object.

Examples

Using the add method

var mySet = new Set();

mySet.add(1);
mySet.add(5).add('some text'); // chainable

console.log(mySet);
// Set [1, 5, "some text"]
