document.plugins

The plugins read-only property of the Document interface returns an HTMLCollection object containing one or more HTMLEmbedElements representing the <embed> elements in the current document.

For a list of installed plugins, use NavigatorPlugins.plugins instead.

Syntax

embedArrayObj = document.plugins

Value

An HTMLCollection, or null if there are no embeds in the document.
