document.URL

The URL read-only property of the Document interface returns the document location as a string.

Syntax

var string = document.URL

Example

JavaScript

document.getElementById("url").textContent = document.URL;

HTML

<p id="urlText">
  URL:<br/>
  <span id="url">URL goes here</span>
</p>
