GlobalEventHandlers.onresize

The GlobalEventHandlers.onresize property contains an EventHandler triggered when a resize event is received.

Syntax

window.onresize = funcRef;

Parameters

  • funcRef is a reference to a function.

Example

window.onresize = doFunc;

<p>Resize the browser window to fire the resize event.</p>

<p>Window height: <span id="height"></span></p>
<p>Window width: <span id="width"></span></p>

<script type="text/javascript">
  var heightOutput = document.querySelector('#height');
  var widthOutput = document.querySelector('#width');

  function resize() { 
    heightOutput.textContent = window.innerHeight;
    widthOutput.textContent = window.innerWidth;
  }

  window.onresize = resize;
</script>
</body>
</html>

Notes

The resize event is fired after the window has been resized.
