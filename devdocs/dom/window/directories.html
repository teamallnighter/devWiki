window.directories

Obsolete since Gecko 2.0 (Firefox 4 / Thunderbird 3.3 / SeaMonkey 2.1)
This feature is obsolete. Although it may still work in some browsers, its use is discouraged since it could be removed at any time. Try to avoid using it.

Returned the window personalbar toolbar object. Use the window.personalbar instead.

Syntax

var dirBar = window.directories;

Parameters

dirBar is an object of the type BarProp.

Example

<script> 
 function dirs() { 
  alert(window.directories);
 } 
</script>

Specification

Not part of specification.
