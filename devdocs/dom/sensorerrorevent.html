SensorErrorEvent

The SensorErrorEvent interface of the Sensor APIs provides information about errors thrown by a Sensor or related interface.

Constructor

SensorErrorEvent.SensorErrorEvent()
Creates a new SensorErrorEvent object.

Properties

SensorErrorEvent.error Read only
Returns the DOMException object passed in the event's contructor.
