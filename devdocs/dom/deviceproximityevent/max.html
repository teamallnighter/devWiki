DeviceProximityEvent.max

This is an experimental technology
Check the Browser compatibility table carefully before using this in production.

The max property provides the maximum sensing distance the sensor is able to report, in centimeters.

Syntax

var value = instanceOfDeviceProximityEvent.max;

Value

A positive number indicating the maximum distance, in centimeters (cm), that the device's proximity sensor is able to detect and report.
