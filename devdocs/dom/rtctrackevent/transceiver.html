RTCTrackEvent.transceiver

The WebRTC API interface RTCTrackEvent's read-only transceiver property indicates the RTCRtpTransceiver affiliated with the event's track. The transceiver pairs the track's receiver with an RTCRtpSender.

Syntax

var rtpTransceiver = trackEvent.transceiver;

Value

The RTCRtpTransceiver which pairs the receiver with a sender and other properties which establish a single bidirectional SRTP stream for use by the track associated with the RTCTrackEvent.

Note: The RTCRtpReceiver referred to by this RTCRtpReceiver's receiver property will always be the same as the RTCTrackEvent's receiver property.
